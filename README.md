# **Cloud File Storage — дипломный проект**

Cloud File Storage — это полноценное клиент-серверное приложение для хранения файлов в облаке, написанное на Java.

Проект состоит из трёх частей:
1. Backend (Spring Boot) — REST API для авторизации, загрузки, скачивания, удаления и переименования файлов.
2. Frontend (JavaScript) — веб-интерфейс для пользователя.
3. PostgreSQL — база данных для хранения пользователей, токенов и данных о файлах.

Для запуска используется **Docker Compose**, который автоматически поднимает три контейнера:
Backend → Frontend → PostgreSQL.

## **Основная функциональность:**

### Авторизация
* **POST** /login — вход, получение токена
* **POST** /logout — выход, инвалидирует токен

### Работа с файлами
* **GET** /file — скачивание файла
* **POST** /file — загрузка файла
* **PUT** /file — переименование файла
* **DELETE** /file — удаление файла
* **GET** /list — список файлов

## **Используемые технологии**
* Java 17
* Spring Boot (Web, Data JPA)
* PostgreSQL
* Docker & Docker Compose
* Mockito - unit-тесты
* Testcontainers - интеграционные тесты

## Запуск через Docker

Проект включает файл **docker-compose.yml**, который поднимает:
* контейнер PostgreSQL
* приложение Spring Boot
* фронтенд приложения

## Как запустить
В корне проекта выполнить:`docker compose up --build`

После запуска сервис будет доступен по адресу: http://localhost:8081

База данных работает на порту: `localhost:5432`

## Инициализация данных

При запуске контейнера автоматически создаются таблицы и добавляется тестовый пользователь благодаря файлу **init.sql**.

### Дополнительно:

Проект строго следует OpenAPI-спецификации, корректно обрабатывает ошибки и выполняет все требования дипломного задания.
